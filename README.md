# Task Manager API

REST API for task management built with Node.js, Express and TypeScript.

## Current Features

* Create tasks
* List all tasks

## Technologies

* Node.js
* Express.js
* TypeScript

## Getting Started

```bash
git clone https://github.com/mardoqueugmartins/task-manager-api.git

cd task-manager-api

npm install

npm run dev
```
